## ⚡️ Dev Guide

1. Clone this repo.

   ```sh
   git clone https://github.com/bagusrizkis/lectureH8.git
   ```

2. Start developing.

   ```sh
   gatsby develop
   # or
   npm run start
   ```

3. Are you ready for launch?

   Your site is now running at `http://localhost:8000`
